# Shark Terminal

Terminal like emulator with user daily activites....
Run Setup for libary installations and binary setup...
Program only for Linux OS & Android

**Run:**

```$ ./setup.sh ``` 

to build and configure tools

**Program File Structure (Architecture)**
```Terminal/
├── LICENSE
├── README.md
├── setup.sh
├── term
├── term.py
├── UTILS/
│   ├── android_setup.py
│   ├── kali_setup.py
│   └── LIBRARY_SETUP/
│       ├── pycryptodome.sh
│       └── pytenable.sh
```

**Note:**
```
1. Tested under Kali LInux Os and Android OS
2. Some tools requires superuser privileg (on feature 8, 9, 12)...
3. For kali installation run setup.sh as root
```

**Features:**
```
1. Binary Tools
2. Port Scanner
3. File Management (Supports Cryptography)
4. Wifi Chat Room
5. File Transfer Via Wifi
6. Reverse Shell
7. Weather Tools
8. Vulnerabilty Scanner
9. Network Information
10. Cpu Information (might work without root access on earlier Android versions)
11. Mobile Number Osint(excluding personal information)
12. Grab Website IP...
```
